## Architecture

**Movie-Muse** (branded as "Movify") is a movie discovery app built on **TanStack Start** (full-stack React with SSR via Nitro). It fetches data from the **TMDB API** using TanStack Query.

### Stack

- **TanStack Start** — full-stack React framework with SSR
- **TanStack Router** — file-based routing (auto-generates `src/routeTree.gen.ts`)
- **TanStack Query** — server state / data fetching
- **Tailwind CSS v4** — styling via CSS custom properties (OKLCH color space)
- **shadcn/ui** (new-york style, slate base) — component library built on Radix UI primitives
- **React Hook Form + Zod** — form state and validation
- **npm** — package manager

### File-Based Routing (`src/routes/`)

| File | URL |
|------|-----|
| `index.tsx` | `/` |
| `about.tsx` | `/about` |
| `users/$id.tsx` | `/users/:id` (dynamic — bare `$`) |
| `posts/{-$category}.tsx` | `/posts/:category?` (optional) |
| `files/$.tsx` | `/files/*` (splat — read via `_splat` param) |
| `_layout.tsx` | layout route (renders children via `<Outlet />`) |
| `__root.tsx` | app shell — wraps every page |

**Do not** create `src/pages/`, `app/layout.tsx`, or other Next.js/Remix patterns. The only root layout is `src/routes/__root.tsx`.

`src/routeTree.gen.ts` is auto-generated by the TanStack Router Vite plugin — never edit it by hand.

### Key Conventions

- **Path alias**: `@/*` maps to `./src/*`
- **`cn()` utility** (`src/lib/utils.ts`): merge Tailwind classes (uses `tailwind-merge` + `clsx`)
- **Prettier config**: 100-char line width, double quotes, trailing commas, semicolons
- **Styling**: Design tokens are CSS custom properties on `:root` / `.dark` in `src/styles.css`; Tailwind utilities consume them via `var(--...)`
